## 🚀 Como usar:

### Forma Automática (Recomendada)
```
1. Usuário acessa /sessoes
2. Conecta WhatsApp (escaneia QR)
3. Sistema AUTOMATICAMENTE:
   ✓ Detecta conexão via webhook
   ✓ Chama WAHA API para listar grupos
   ✓ Extrai membros de cada grupo
   ✓ Cria contatos para membros
   ✓ Salva tudo no banco
4. Usuário acessa /grupos
5. VÊ todos grupos extraídos com membros!
```

### Forma Manual (Se Necessário)
```
1. /grupos → Seleciona sessão
2. Clica "Forçar Extração de Grupos"
3. Aguarda 5-10 segundos
4. Grupos atualizados!
```

## 📈 API Endpoints

```
GET    /api/grupos                          Listar todos
GET    /api/grupos/{id}                     Detalhes de um
GET    /api/grupos/{id}/members             Membros
POST   /api/grupos/session/{id}/extract     Forçar extração
POST   /api/grupos/{id}/re-extract          Re-extrair
POST   /api/grupos/{id}/add-to-campaign     Adicionar campanha
DELETE /api/grupos/{id}                     Deletar
```
